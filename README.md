Extensão de navegador que salva automaticamente as notícias publicadas no site do IFBA Ilhéus, permitindo acesso rápido ao conteúdo mesmo offline. Compatível com Google Chrome e Microsoft Edge.

## ✨ Funcionalidades

- ✅ Salva notícias do site do IFBA Ilhéus automaticamente
- ✅ Acesso rápido via barra de extensões do navegador
- ✅ Interface limpa e intuitiva
- ✅ Compatível com Google Chrome e Microsoft Edge
- ✅ Funciona sem precisar abrir o site

## 🛠️ Tecnologias

| Tecnologia | Uso |
|---|---|
| JavaScript | Lógica da extensão e manipulação do DOM |
| HTML | Interface do popup |
| CSS | Estilização |
| Chrome Extension API | Manifest v3, content scripts |

## 📦 Como instalar

### No Google Chrome ou Microsoft Edge:

```bash
# 1. Clone o repositório
git clone https://github.com/AlanPrates/Extensao-IFBA-Ilheus.git

# 2. Abra o Chrome e acesse
chrome://extensions/

# 3. Ative o "Modo do desenvolvedor" (toggle no canto superior direito)

# 4. Clique em "Carregar sem compactação"

# 5. Selecione a pasta do projeto clonado
```

## 📂 Estrutura

```
Extensao-IFBA-Ilheus/
├── assets/        # Ícones e imagens
├── css/           # Estilos do popup
├── webfonts/      # Fontes
├── index.html     # Interface do popup
├── script.js      # Lógica principal
└── manifest.json  # Configuração da extensão
```
